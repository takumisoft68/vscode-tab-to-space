# Tab to Space

Convert all tabs to indentation using spaces.

## 1. Features

| Title                                         | Command          | Default Keybinding | In the Editor Right Click Menu |
| :-------------------------------------------- | :--------------- | :----------------- | :----------------------------- |
| Convert all tabs to indentation using spaces. | tabspace.convert |                    | No                             |
